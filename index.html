Sopa de Letras

Selecciona las letras para encontrar las palabras. Usa el teclado o el ratón.

La cuadrícula se genera con JavaScript.

Palabras a encontrar

La lista de palabras aparecerá aquí.

Instrucciones

  1. Busca las palabras de la lista dentro de la cuadrícula.
  2. Selecciona letras consecutivas para marcar una palabra.
  3. Puedes usar el botón "Resolver sopa de letras" para mostrar las soluciones.
  4. Usa el botón "Reiniciar la sopa" para generar una nueva versión (o recargar la página).
